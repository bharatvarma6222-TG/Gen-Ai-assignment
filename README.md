# Gen AI Internship Assignment – Synlabs

## Overview
This repository contains solutions for a 3-part Gen AI internship assignment:
1. AI Video Generation Tool
2. AI-Based SEO Blog Creation Tool
3. High-Level to Low-Level Architecture Automation Tool

---

## Task 1: AI Video Generator
### Features
- Scrapes trending news
- Generates script automatically
- Extracts topic-based images from articles
- Creates a 30–60 sec MP4 video with text overlays

### Tech Stack
Python, MoviePy, Pillow, Google News RSS

---

## Task 2: SEO Blog Generator
### Features
- Scrapes trending products
- Generates SEO keywords automatically
- Creates a 150–200 word blog post
- Export-ready for Medium / WordPress

### Tech Stack
Python, BeautifulSoup, Google Autocomplete

---

## Task 3: Architecture Generator
### Features
- Converts high-level requirements into:
  - Modules
  - APIs
  - Database schema
  - Pseudocode
- Domain-aware with fallback logic

### Tech Stack
Python
